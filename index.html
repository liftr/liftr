    OR...

    Download and configure any repository

    1. Install the repository as a component

      $ liftr install someguy/awesome-component

    2. Add a _lift.scss file to the components/awesome-component folder to tell Liftr which files it should lift. It should look like this:

      // Requires another liftr component
@import "liftr/jquery";

// Requires this component sass/css file
@import "css/awesome-component";

// Tells Liftr to include this javascript to lift this component
// Note that you don't have '.js' or '.min.js' extension. Liftr will choose appropriately.
@include load-js("awesome-component", "js/awesome-component");

// once the everything is defined, let Liftr know it is all done
@include lifted("bootstrap");

    3. That's it! Have fun!

      Now you can simply add @import 'lift/awesome-component' in your sass application and Liftr will do the heavy lifting for you.
